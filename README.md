
# 各地历年公积金基数

- 所有数字单位为人民币元
- 只统计基数上限，一般解释为3倍当地社会平均工资
- 年份代表含义为：本年度变更月份到下一年变更月份之间的基数。  
  比如，2023年广州，指的是广州2023.07（变更月份）-2024.06期间的公积金基数上限。
